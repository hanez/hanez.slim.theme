# Installation

```
git clone https://git.xw3.org/hanez/hanez.slim.theme.git
sudo cp -r hanez.slim.theme /usr/share/slim/themes/hanez
```

Then you can set the following in /etc/slim.conf
```
current_theme       hanez
```
